# Task Data Model

The Task entity represents an academic activity that a student needs to manage.

Attributes:

task_id
Unique identifier for each task.

title
The name of the academic task.

course
The course related to the task.

task_type
Defines the category of the task (assignment, exam, quiz, self-study).

deadline
The due date of the task.

estimated_study_hours
The estimated amount of time required to complete the task.

priority
Priority level determined based on deadline proximity.

status
Indicates whether the task is pending or completed.
